Image Editing

Images can be added to any of your pages to create visual interest and help break up text-heavy courses. This Images page illustrates the different ways images can be used.

You can do one of the following:

  • Replace the images and text on this page
  • Follow the directions below to insert new images on other pages
  • Copy the image alignment samples from this page and past them to other pages

Replace Image

This template comes with some image placeholders. To replace the image:

  1. Select the image and delete it (Note: Do not move your cursor)
  2. Click the Insert Image icon
  3. Select the source of your image
  4. Click the Choose Destination button to save it to the folder you want

Recommendation for banner image size: at least 1200 pixels wide and 400 pixels high.

Insert New Image

Inserting your own image is simple. Once in the Edit HTML mode, you can use Insert Image to add an image. If the image is not the right size, you can edit it using the inline image editor.

Inline image editing: the HTML editor now (after February 2018/10.7.10 release) has advanced image editing features, including resizing, flipping, rotating, and adjusting brightness, sharpness and color.

Note: the banner images have a CSS style rule that makes these images full page width. So, resizing them will have no impact.

Image Alignment

The HTML editor provides Align Left, Align Right and Align Full functions to place an image in particular location.

For example, if you want your image to appear on the left with the text wrapping around it:

  1. Insert your image on the empty line before the paragraph
  2. Crop the image or set its dimension to your desired size
  3. Apply Align Left on the selected image
