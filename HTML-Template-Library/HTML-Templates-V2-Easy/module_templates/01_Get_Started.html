Get Started

Welcome to the Daylight HTML template. This template was designed to help non-developers create clean yet professional content pages using Brightspace HTML Editor.

How to Use HTML Template

Step 1. Deploy HTML Templates

You can add this template package to a specific course using Manage Files tool, or to share it across your Brightspace in the Shared Files tool.

Step 2. Enable HTML Templates

Itâ€™s a good practice to set the template location for your course. This saves time from browsing to the same folder repeatedly.

To enable the HTML template in the content tool, please check the Brightspace Help: Enable HTML Templates in Content.

Step 3. Apply a HTML Template

This template set comes with several purpose-built topic types:

  • Course Introduction
  • Module Introduction
  • Meet Your Facilitator
  • Basic page
  • Elements page
  • Image page
  • Video Lecture
  • Conclusion pages.

You can replace the content on each of these pages to create responsive and accessible topics in your course.

To apply an HTML template to a new content page, please check the Brightspace Help: Apply an HTML template to a new document in Content.

Note: Apply the template to an existing page will overwrite the content and you will lose your work. Click Cancel to discard the changes.
