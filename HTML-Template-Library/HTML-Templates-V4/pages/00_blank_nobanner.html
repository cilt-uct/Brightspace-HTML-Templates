Easy Editing Use the HTML Editor

You can use the HTML editor to make quick and easy changes without needing any prior knowledge of HTML. Enter your content and use the available controls to apply formatting to your text.

Copying Text

When writing content, it is a great practice to first write content in a document, such as Microsoft Word. It allows stakeholders to easily collaborate and track changes to content. It also allows you to spot spelling and grammar errors early on.

When pasting text from a Word document into the HTML editor, however, some of the document's text styling will copy over. This will clash with the styles that are carefully crafted for this template. So, we recommend to paste the text without formatting using Ctrl + Shift + V (Windows) or Cmd+Shift+V on Mac.
