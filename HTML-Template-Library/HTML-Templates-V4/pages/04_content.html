This page includes samples of learning materials accompanied by instructional text. Instructional text describes how a reading, video or image should be engaged with by the student, for example by indicating the key concept it discusses or illustrates, or by providing some questions for the students to bear in mind as they engage with the learning resource.

  Video Title

The Video layout includes an embedded YouTube video, which can scale in any browser or mobile device. Embedding videos in the page is a great way to present video content, while accompanying it with supporting context, explanations and activities.

Direct Link: Create a File and Insert Stuff(this link opens in a new window/tab)

To replace the video,

  1. Delete the video, and leave the cursor in-place. To do so:
    1. click on the video and pause it
    2. carefully click the area preceding (left of) the video within the colored video container
    3. hit the Delete key
  2. Click the Insert Stuff icon
  3. Do one of the following:
    • Select Course Offering Files to insert a file you have already uploaded to your course;
    • Select Add Video Note to record and publish a webcam video of yourself;
    • Select YouTube, search for your video; or
    • Select Enter Embed Code, paste in the YouTube video embed code

Images

Images can be added to any of your pages to create visual interest and help break up text-heavy courses. This section illustrates the different ways images can be used. You can insert images from your course files and provide alternative text for students using assistive software, and align the images to the left, right or full page.

Insert New Image

Inserting your own image is simple. Once in the Edit HTML mode, you can use Insert Image to add an image. If the image is not the right size, you can edit it using the inline image editor.

When you insert an image, you will be prompted to add alternative text. This text helps students using assistive software to understand the pedagogical purpose of the image, or if the image is purely decorative.

Aligning images

The HTML editor provides Align Left, Align Right and Align Full functions to place an image in particular location.

For example, if you want your image to appear on the left with the text wrapping around it:

  1. Insert your image on the empty line before the paragraph
  2. Crop the image or set its dimension to your desired size
  3. Apply Align Left on the selected image
